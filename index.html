完整使用教學

從 ER 概念圖到可執行 SQL，三個階段全覆蓋

1

放置節點

點擊左側工具列的元素類型（實體、弱實體、關聯…），畫布游標變成 ✛，再點擊畫布任意位置即可放置。

實體 Entity
弱實體 Weak Entity
關聯 Relationship
屬性 Attribute
多值屬性 Multi-value
衍生屬性 Derived
聚合 Aggregation
2

選取節點與工具列

點擊節點後浮現工具列，含以下操作：

連線點另一個節點完成連線；再點同節點取消
新增選類型 → 點畫布放置，自動連回原節點
複製在附近複製一個相同節點
🔓 鎖定鎖定後無法拖動，防止意外移位；再按解鎖
刪除刪除節點及其所有連線
3

編輯節點文字

雙擊節點進入文字編輯模式，直接修改標籤名稱。

頂部工具列可設定選取節點的：字級大小、字型、粗體、斜體、顏色。

4

連線與基數標記

右鍵點擊連線開啟選單，可刪除連線或設定基數標記。

基數標記顯示在連線兩端靠近節點處，可輸入任意文字（1、n、m…）或選快速預設：

1:1 1:n n:1 m:n
5

畫布操作

縮放滾輪（Trackpad 雙指）
平移畫布拖拉空白處
全覽點頂部「全覽」按鈕
移動節點拖拉節點
多選節點框選（拖拉空白處時按住 Shift）
6

匯出與分享

匯出 SVG右上角「匯出 SVG」按鈕，下載向量圖
分享連結右上角「分享」→ 設定唯讀或可編輯 → 複製連結
即時協作多位使用者同時編輯，頭像顯示在線上人員
1

建立邏輯模型

在 Home 頁面點擊「新增邏輯模型」，或在已有 ER 圖中點「連結 ER 圖 → 轉換」，自動依 ER 實體產生資料表與欄位。

2

表格操作

新增表格點擊左側「資料表」後放置到畫布
改名雙擊表格標題，或點工具列「改名」
移動拖拉表格標題列
刪除表格點工具列「刪除」或按 Delete
3

欄位操作

點擊欄位列 → 右側出現欄位工具列：

PK設為主鍵
FK標記外鍵
← →在當前欄位前後插入新欄位
連線建立 FK 箭頭連結至另一表格欄位
改名雙擊欄位或點「改名」直接輸入
刪除刪除該欄位（至少保留一欄）
4

連結 ER 圖自動轉換

點右上角「連結 ER 圖」→ 選擇目標 ER 圖 → 點「從 ER 圖轉換」，系統自動：

  • 以每個 ER 實體建立對應資料表
  • 實體屬性自動成為欄位
  • 已存在同名表格不重複建立
1

表格結構（三欄式）

每個資料表顯示三欄：

欄位名稱型別說明
PK id INT 主鍵
FK dept_id INT 外部鍵
name VARCHAR 建立索引
2

欄位屬性設定

點擊欄位 → 右側工具列設定：

PK設為主鍵（黃色，自動帶底線）
FK標記外鍵（藍色）
NN非空約束（欄位名底線）
型別INT / BIGINT / VARCHAR / CHAR / TEXT / FLOAT / DECIMAL / DATE / DATETIME / TIMESTAMP / BOOLEAN / BLOB
說明自由輸入：主鍵、外部鍵、建立索引[索引:欄位名]…
3

FK 連線

建立外鍵關聯箭頭有兩種方式：

  1. 點選表格 → 工具列「FK 連線」→ 點來源欄位 → 點目標欄位
  2. 點選欄位 → 工具列「連線」→ 點目標表格欄位

點擊 FK 箭頭可刪除連線。

4

匯出 SQL DDL

點右上角「匯出 SQL DDL」，選擇方言後即時產生建表語法：

MySQL
AUTO_INCREMENT
COMMENT '主鍵'
PostgreSQL
SERIAL / BIGSERIAL
COMMENT ON COLUMN
SQLite
VARCHAR → TEXT
-- 說明 (inline)
SQL Server
IDENTITY(1,1)
NVARCHAR / BIT
5

從邏輯模型匯入

點右上角「連結邏輯模型」→ 選取目標邏輯模型 → 點「從邏輯模型匯入」，系統自動：

  • 以邏輯模型每張表格建立實體資料表
  • 同步 PK / FK 欄位屬性
  • 自動填入「說明」欄（主鍵/外部鍵）
  • 已存在同名表格不重複建立
