使用教學

ERCanvas 完整操作說明

1
放置節點

點左側工具列選元素類型 → 游標變成 ✛ → 點畫布放置。

▭ 實體  ▭▭ 弱實體  ◇ 關聯  ◇◇ 弱關聯
◯ 屬性  ◯◯ 多值屬性  ⌀ 衍生屬性  ⬜◇ 聚合
2
選取節點工具列
連線點另一節點完成；點同節點取消
新增選類型放置，自動連回原節點
複製在附近複製相同節點
🔓鎖定鎖定後不可拖動，再按解鎖
刪除刪除節點與所有連線
3
編輯文字 & 樣式
• 雙擊節點進入文字編輯
• 頂部工具列：字級 / 字型 / 粗體 / 斜體 / 顏色
• Enter 或點其他處確認，Esc 取消
4
基數標記
• 右鍵點連線 → 設定基數標記
• 標記顯示在連線兩端靠節點處
• 快速選：1:1 · 1:n · n:1 · m:n
• 或自行輸入任意文字
5
畫布操作
• 縮放：滾輪 / 雙指
• 平移：拖拉空白處
• 全覽：頂部「全覽」按鈕
• 鎖定節點：選取後點工具列鎖定
6
匯出與分享
• 匯出 SVG：右上角按鈕
• 分享：設定唯讀或可編輯連結
• 即時協作：多人同時編輯，頭像顯示在線狀態
1
建立邏輯模型
• Home 點「新增邏輯模型」手動建立
• 或點「連結 ER 圖 → 從 ER 圖轉換」自動生成
• ER 實體 → 資料表，屬性 → 欄位
2
表格操作
• 新增：點左側「資料表」→ 點畫布放置
• 改名：雙擊表格標題 或 工具列「改名」
• 移動：拖拉表格標題列
• 刪除：工具列「刪除」或 Delete 鍵
3
欄位操作
PK設為主鍵
FK標記外鍵
← →前後插入新欄位
連線建立 FK 箭頭至另一表格
改名雙擊欄位 或 點「改名」
刪除至少保留一欄
4
連結 ER 圖自動轉換
1. 右上角「連結 ER 圖」
2. 選取目標 ER 圖
3. 點「從 ER 圖轉換」
• 已存在同名表格不重複建立
1
三欄式表格結構
欄位名稱型別說明
PK id INT 主鍵
FK dept_id INT 外部鍵
name VARCHAR 建立索引
2
欄位屬性設定
PK主鍵（粗體，自動 NOT NULL）
FK外鍵標記
NN非空（欄位名底線）
型別INT / VARCHAR / TEXT / DATE…
說明主鍵、外部鍵、建立索引…
3
FK 連線
方式①：點表格 → 工具列「FK連線」→ 點來源欄位 → 點目標欄位
方式②：點欄位 → 工具列「連線」→ 點目標表格欄位
• 點擊 FK 箭頭可刪除連線
• Esc 取消連線操作
4
匯出 SQL DDL
右上角「匯出 SQL DDL」→ 選方言：
MySQL
AUTO_INCREMENT
PostgreSQL
SERIAL / BIGSERIAL
SQLite
VARCHAR→TEXT
SQL Server
IDENTITY(1,1)
